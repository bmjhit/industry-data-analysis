# A 股行业与基金投资看板

这是一个本地静态看板原型，用于按日、周、月、季度、年度观察 A 股行业表现，并把行业强弱、估值、资金、风险与基金投资赛道建议联系起来。

## 当前版本

- 纯前端静态页面，无需安装依赖。
- 使用 `data/industry-sample.json` 中的示例数据。
- 支持行业总览、周期切换、行业详情、基金赛道建议、组合配置建议。
- 所有输出均为研究辅助，不构成保证收益或确定性投资建议。

## 使用方式

直接在浏览器打开：

```text
index.html
```

或者使用任意静态服务器托管当前目录。

## 后续数据接入方向

- A 股股票与行业行情：东方财富、同花顺、聚宽、Tushare、AkShare 等数据源。
- 指数与行业分类：中证指数、申万行业、证监会行业分类。
- 基金数据：基金净值、重仓股、行业配置、基金经理、费率、规模、回撤、同类排名。

## 投资原则

- 先做资产配置，再选具体基金。
- 核心仓优先宽基和低费率指数基金。
- 行业主题基金作为卫星仓，控制比例和回撤。
- 主动基金重点观察基金经理任期、风格稳定性、回撤控制和长期超额收益。
