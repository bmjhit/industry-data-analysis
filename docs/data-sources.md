# 数据源规划

## 行情与行业

- 行业分类：申万一级/二级行业、中证行业、证监会行业分类。
- 股票行情：开高低收、成交额、换手率、市值、涨跌幅。
- 行业指标：行业涨跌幅、上涨家数占比、成交额占比、估值分位、盈利预测。
- 可选工具：AkShare、Tushare、东方财富公开数据、聚宽或 Wind 等商业数据。

## 基金

- 基金基础信息：基金代码、名称、类型、成立时间、规模、费率、跟踪指数。
- 基金表现：近 1 周、1 月、3 月、6 月、1 年、3 年收益，最大回撤，波动率。
- 基金持仓：前十大重仓股、行业配置、债券比例、现金比例。
- 基金经理：任期、管理规模、历史回撤、风格稳定性。

## 推荐逻辑

1. 先基于风险偏好生成大类配置比例。
2. 用行业评分筛选赛道，过滤高估值高拥挤方向。
3. 每个赛道优先选低费率、规模适中、跟踪误差低的指数基金。
4. 主动基金需额外考察基金经理任期、风格漂移、回撤控制和长期超额收益。
5. 输出建议等级：核心配置、卫星配置、观察、暂不配置。

## 风险提示

行业评分和基金筛选只适合辅助研究，不构成收益承诺。主题基金净值波动较大，建议配合定投、再平衡和最大回撤约束使用。
