# 数据源规划

## 行情与行业

- 当前实现：AkShare 的东方财富行业板块接口。
- 行业列表：`stock_board_industry_name_em`
- 行业历史行情：`stock_board_industry_hist_em`
- 行业成分股：`stock_board_industry_cons_em`
- 已计算指标：日/周/月/季/年收益、上涨家数占比、换手热度、趋势分、资金动量、近一年回撤风险。
- 注意：东方财富公开页面可能限流或短暂失败，脚本采用重试和原子写入，失败时不会覆盖旧数据。

## 基金

- 当前实现：AkShare 的东方财富开放式基金排行接口。
- 基金排行：`fund_open_fund_rank_em(symbol="全部")`
- 已使用字段：基金代码、基金简称、近 1 月、近 3 月、近 1 年、手续费。
- 当前候选逻辑：按行业关键词匹配基金名称，并优先展示近 3 月表现靠前的候选。
- 后续应补充：基金规模、基金经理任期、跟踪指数、持仓行业、最大回撤、波动率、同类排名。

## 推荐逻辑

1. 先基于风险偏好生成大类配置比例。
2. 用行业评分筛选赛道，过滤高估值高拥挤方向。
3. 每个赛道优先选低费率、规模适中、跟踪误差低的指数基金。
4. 主动基金需额外考察基金经理任期、风格漂移、回撤控制和长期超额收益。
5. 输出建议等级：核心配置、卫星配置、观察、暂不配置。

## 风险提示

行业评分和基金筛选只适合辅助研究，不构成收益承诺。主题基金净值波动较大，建议配合定投、再平衡和最大回撤约束使用。
